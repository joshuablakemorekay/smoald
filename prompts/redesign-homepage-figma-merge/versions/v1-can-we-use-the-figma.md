# v1 — Can we use the Figma?

```
[Figma Make link] - I like this Figma design for https://smoald.com/ - can we use it?
```

## Context

Opening ask. The link was a **Figma Make** project (AI-generated React/Vite/Tailwind code), which can't be viewed without a login. The `package.json` was later shared, revealing a heavy framework export with dozens of UI libraries.

## What changed after this round

Established the key decision: **rebuild the design's look inside the existing static site** rather than adopt the React framework. This keeps smoald.com fast, build-step-free, and consistent with its sibling pages — and makes it easy to blend in only the parts the user liked.
